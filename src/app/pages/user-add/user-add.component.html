Form Generator Usage Example

This form is generated dynamically using the FormGeneratorComponent. Key features demonstrated:

  • Responsive Layout: Fields are col-6 on desktop, col-12 on mobile (unless fullWidth is set)
  • Multiple Input Types: text, email, password, number, date, select, textarea, checkbox, radio
  • Validation: Required fields, email validation, min/max length, pattern matching
  • Styling Options: default, solid, transparent styles
  • Event Handling: Form submission, reset, real-time changes, individual field changes
  • Initial Data: Pre-populate fields with default values
